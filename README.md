# KIIT ERP - Student Attendance & Management System (Prototype)

This repository contains the frontend prototype for the **KIIT University Enterprise Resource Planning (ERP)** system. It is designed to demonstrate the user interface and workflow for the HOD and Faculty modules, focusing on student attendance and academic management.

* **Project Name:** ADLABPROJECT (Application Development Lab)
* **Type:** Frontend Prototype (UI/UX)
* **Theme:** KIIT University Branding (Green/White)

## ‚ú® Key Features
* **Interactive Dashboard:** Real-time visualization of total students, staff, and courses.
* **Smart Navigation:** Fully functional sidebar with toggleable sections (Home, Add Student, Manage Staff, etc.).
* **Responsive Design:** Built on **AdminLTE 3** (Bootstrap 4), ensuring compatibility across mobile and desktop devices.
* **Form Simulation:** "Add Student" and "Add Staff" forms with frontend validation and success alerts.
* **Data Visualization:** Visual progress bars for attendance metrics across different engineering branches.
